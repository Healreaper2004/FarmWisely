\# 🌱 FarmWisely



AI-powered farming intelligence platform for yield prediction and soil analysis.



\## 🚀 Features

\- Crop yield prediction using ML

\- CSFI (Soil Fertility Index) calculator

\- Sustainability metrics (WPI, NES, III, SRS)

\- Smart recommendations (fertilizer, irrigation, crop rotation)

\- Privacy-preserving case storage



\## 🛠️ Tech Stack

\- Frontend: HTML, CSS, JavaScript

\- Backend: Flask (Python)

\- ML: Scikit-learn

\- Database: MongoDB
